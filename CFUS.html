CFU 1

CFU 1, which was our first ever check for understanding was a review on the concepts of using user input. In this CFU, users would input their first name and last name, and the program would output hello and their name

CFU 2

CFU 2, similar to CFU 1, is a review of using user input in a program to output a sentence. Users can input their first name, last name, their age, and they would respond to the prompt'how long have you been coding'. Based on what the user inputs, the output would print hello with their name, age and the response to the prompt.

CFU 3

In CFU 3, I did a basic review of user inputs and using the skills I learned about numerical operators,I created a program where the user inputs their first and last name, which would output a greeting to them, and they will input a whole number and a decimal number. Using those 2 values, the program will calculate the sum, difference, product, quotient, and remainder and print out each value to the user.

CFU 4

In CFU 4, I created a program where the user can input 3 numbers of their choice and the program will calculate the roots of their equation using the quadratic formula using import math from the python library.
